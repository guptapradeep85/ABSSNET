# Data Availability

The retinal fundus images analysed in this study are third-party data and are not redistributed by the authors. All experiments used the publicly available Diabetic Retinopathy 224x224 Gaussian-Filtered dataset, which can be downloaded directly from Kaggle at <https://www.kaggle.com/datasets/sovitrath/diabetic-retinopathy-224x224-gaussian-filtered>; it is a pre-processed derivative of the APTOS-2019 Blindness Detection dataset (<https://www.kaggle.com/competitions/aptos2019-blindness-detection>). Both remain subject to the original providers' terms of use, and the image files are therefore not mirrored in our repository. The Messidor-2 images used for the zero-shot external evaluation are distributed by ADCIS under a separate registration and licence agreement (<https://www.adcis.net/en/third-party/messidor2/>) and likewise cannot be redistributed.

All derived data generated by this study are publicly available without restriction at https://github.com/guptapradeep85/ABSSNET. These comprise the pooled out-of-fold predicted probabilities, predicted labels, ground-truth labels and reliability scores for all 3,662 images; the cross-validation summary and per-fold metric tables; the per-class out-of-fold metric tables; the calibration and reliability-bin tables; the ablation and external-validation result tables; the frequency-attenuation measurements; the class-distribution counts; the trained per-fold model checkpoints; the experiment logs; and the source files for every figure and table reported in the article. Scripts and documentation that reconstruct the experimental partitions and preprocessing pipeline from the publicly downloadable source images are provided in the same repository.

## Classification of all study materials

### 1. Original public dataset files - NOT redistributed
| Item | Status |
|---|---|
| DR224 Gaussian-filtered fundus images (3,662) | Third-party. Download from Kaggle. |
| APTOS-2019 raw images | Third-party. Kaggle competition, rules acceptance required. |
| Messidor-2 images (1,744) | Third-party. ADCIS registration + licence agreement required. |

### 2. Access procedure
1. **DR224** - create a free Kaggle account, open the dataset page, click *Download*. No approval step. Unpack to `data/DR224/Train_image/<class>/`.
2. **APTOS-2019 raw** - Kaggle account, open the competition page, accept the competition rules, then download. Needed only for the raw-vs-Gaussian ablation.
3. **Messidor-2** - complete the ADCIS request form at <https://www.adcis.net/en/third-party/messidor2/> and accept the licence. Grading labels are distributed separately by the Messidor-2 community. Needed only for the external-validation experiment.

### 15. Files that cannot legally be redistributed
Only the raw fundus image files (DR224 / APTOS-2019 / Messidor-2). **No patient-identifiable data exists in this study** - all three datasets are already de-identified by their providers, and no new human-subject data was collected. Nothing else is withheld.

## Does this satisfy the editor's request to make all available data public?

**Yes.** Every artefact the authors are legally permitted to publish is public: all source code, all configurations, all derived predictions and probabilities, all result tables, all figures, all logs, and all trained model weights. The only withheld items are the third-party image files, whose redistribution is restricted by the original providers' terms - not by author choice. Because the images are freely downloadable by any reader from those providers, and because the split and preprocessing logic is fully deterministic and released, the complete experimental pipeline is reconstructible end-to-end.
